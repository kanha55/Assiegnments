2. What is the notation for following functions?

1.  int f(int a, float b)
    {
        /* Some code */
    }

2.  int f(a, b)
    int a; float b;
    {
        /* Some code */
    }
A.	1. KR Notation 
2. ANSI Notation
B.	1. Pre ANSI C Notation 
2. KR Notation
C.	1. ANSI Notation 
2. KR Notation
D.	1. ANSI Notation 
2. Pre ANSI Notation

Answer: Option C

Explanation:

KR Notation means Kernighan and Ritche Notation.
